#### Q. No: 16

##### Question
To get a string from a given string where all occurrences of its first char have been changed to '$', except the first char itself.

##### Output
```
Enter路input:路restart
resta$t
```
```
Enter路input:路python
python
```

##### Solution
```
str = input("Enter input: ")
ch = str[0]
new_str = ch
new_str += str.replace(ch, '$')[1:]
print(new_str)
```
